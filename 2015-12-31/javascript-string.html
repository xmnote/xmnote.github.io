Javascript String() 方法可把对象的值转换为字符串。该方法与各个对象对应的 toString() 方法的返回值是一样的。

String() 方法说明

定义

Javascript String() 方法可把对象的值转换为字符串。该方法与各个对象对应的 toString() 方法的返回值是一样的。

使用方法

String(object)

参数说明

参数 说明
object 必须参数。要被转换的对象。

浏览器支持

IE Firefox Chrome Safari Opera
